First steps
Get the files

To get started, you'll need to have Sass running. You can download those files here. Then, download the Stacks framework here. Open Sass in a code editor and place the Stack files within the Sass folder.

Hook it up

In your style.scss file, add the following import statements at the top of the document: @import "stacks/_variables"; @import "stacks/_custom"; Anything you write below these two statements will overwrite the Stacks classes. Next, compile your entire Sass folder with a preprocessor to align all code.

Know where to find things

To access global variables, go to _variables.scss Any modifications will directly compile to style.scss. The _custom.scss file is annotated with class types, intended for you to split up into partials as desired.

Grid
Container

The .container class will center and flex your other elements. Make this the outer element in your HTML.

Flex area

The .flex-area class nests inside the .container class. It provides a flex property and formatting that allows for mobile reponsiveness. It can be combined with the .expand-all-children-md or .expand-all-children-sm class to bring smaller elements to 100% width, depending on the breakpoint.

Spread

When building in Stacks,the bricks we stack up are all within the spread class. select between 1 and 6 columns which your spread will occupy. This is ideally combined with the .container , .flex-area , and .expand-all-children- classes in a nesting pattern in order to optimize responsiveness across devices. Examples below:

Panels
Formatting

Panels nest within the grid to diferentiate areas of the screen. Stacks comes with three out-of-the-box options: .default-panel-color , .light-panel-color , and .dark-panel-color .

Text
Type scale

Text size can scale across desktop and mobile devices using the scale class. .p-scale will scale for paragraph elements, .h1-scale will scale for h1 elements, etc. Adjust the scaling effect by modifying the --fontScale CSS custom variable or adjust the size of the text directly by modifying the --fontBaseSize CSS custom variables, both of whic are located in the _custom.scss file.

Body, title

Stacks comes with set .body and .title classes to get you up and running quickly.

Typography

Stacks comes with a ready-to-go typography set. Adjust these default fonts in the _variables.scss file to affect classes such as .body and .title .

Navigation
Nav bar

The nav bar is stripped down to its barest elements -- area and text. Declare the .nav class on the parent element, ideally with a responsive container like .spread-6 . Then add nav text inside of a tags.

Images
Sizing

Images can leverage four responsive classes. .img-full expands to fill the maximum image size possible according to the parent container. .img-lg , .img-md , and .img-sm start out at 500px, 300px, and 150px, respectively. All sizes scale down at the point of clipping.

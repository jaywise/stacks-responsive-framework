First steps
Get the files

To get started, you'll need to have Sass running. You can download those files here. Then, download the Stack framework here. Open Sass in a code editor and place the Stack files within the Sass folder.

Hook it up

In your style.scss file, add the following import statements at the top of the document: @import "stacks/_variables"; @import "stacks/_custom"; Anything you write below these two statements will overwrite the Stacks classes.

Know where to find things

To access global variables, go to _variables.scss Any modifications will directly compile to style.scss. The _custom.scss file is annotated with class types, intended for you to split up into partials as desired.
